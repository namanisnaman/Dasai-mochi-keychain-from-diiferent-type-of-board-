# Dasai-mochi-keychain-from-diiferent-type-of-board-
In this repository, you can find different methods of making Dasai-mochi with different types of boards like esp. 32, esp 32 c3 mini, Arduino Nano and Arduino Uno. All the code is available above; you can easily checkout there, and the connections are available.
# ways to make a dasai mochi
So as we know, the most common way is to make it by using a small OLED and a small-sized ESP-32 C3 mini, but the problem starts when you want an OLED of a bigger size, like 1.3 inch oled whuch is not supported by Huykhong (https://www.themochi.huykhong.com/) or sometimes when you want to make it but you don't have the ESP-32 C3 mini board. But I solved this problem by making alternatives to it. I have given the code to make it with all the boards I have mentioned above

RoboEyes-main.zip -  Arduino Nano and ardiuno uno ( you can download it and then go to Arduino IDE and go to Sketch, include library and select this zip.
